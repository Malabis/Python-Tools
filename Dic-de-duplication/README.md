# python字典去重工具

# Introduce

在渗透测试中必不可少的会遇到爆破时字典太多太冗余的时候，这个时候就需要整理自己字典来更高效的进行工作

## Usage

`python Dic-de-duplication.py filename1 filename2 newfilename`

filename1和filename2是所需要整理的两个原字典，newfilename是整理完之后的自己设定的新的字典名
